# LocalAlpha Releases

**The privacy-first local dashboard for QuantConnect LEAN backtests.**

Visualize equity curves, analyze trade execution, and debug logs without cloud uploads.

## Features
- **Local & Private:** Your strategy code and results never leave your machine.
- **Interactive Charts:** High-performance canvas charts for equity, drawdown, and benchmark comparison.
- **Advanced Metrics:** Recalculate Sharpe, Sortino, PSR, and Calmar locally.
- **Trade X-Ray:** MAE/MFE analysis to optimize stop-losses.
- **Log Explorer:** Regex-powered search for gigabyte-sized log files.

## Download
Download the latest version from the [Releases Page](https://github.com/LocalAlpha-io/localalpha-releases/releases).

- **macOS:** Universal Binary (.dmg)
- **Windows:** Installer (.msi)
- **Linux:** AppImage / Deb

## Auto-Update
This repository hosts the `update.json` file used by the LocalAlpha application to detect and install updates automatically.
